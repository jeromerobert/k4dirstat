A.3. Compilation and installation

In order to compile and install KApp on your system, type the following in the base directory of the Icon Editor distribution: 
 % ./configure
 % make
 % make install

Since KApp uses autoconf and automake you should have not trouble compiling it. Should you run into problems please report them to the KDE mailing lists.
